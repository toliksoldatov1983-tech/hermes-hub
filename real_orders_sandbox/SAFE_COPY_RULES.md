# Safe Copy Rules

Date: 2026-06-17

## Что такое safe copy

Safe copy — это копия (не оригинал) данных заказа, подготовленная пользователем вручную.

## Правила

1. Safe copy ≠ оригинал заказа
2. Hermes НИКОГДА не читает оригиналы
3. Hermes НИКОГДА не сканирует рабочие папки
4. Hermes НИКОГДА не открывает `E:\Заказы...`
5. Только пользователь кладёт файлы в `input_safe_copies/`
6. Все результаты — `not_final` / `review_required`

## Что удалить из safe copy перед копированием

- Личные данные клиентов (имена, телефоны)
- Пароли / ключи
- Финансовую информацию
- Файлы Corel/ArtCAM/CNC (.cdr, .art)
