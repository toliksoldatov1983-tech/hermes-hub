```

```

## Что можно

- Описание заказа (текст)
- Размеры, материал, цвет
- Тип работы
- Примечания

## Что нельзя

- .cdr / .art / CNC файлы
- Личные данные клиентов
- Финансовую информацию
- Оригиналы (только копии)
