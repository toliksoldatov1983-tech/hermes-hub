# Safe Copy Naming Rules

Date: 2026-06-17

## Preferred

```
ORDER_SAFE_COPY_002.md
ORDER_SAFE_COPY_003.md
```

## If Windows adds .txt

```
ORDER_SAFE_COPY_002.md.txt  ← допустимо (.txt разрешён)
```

## How to save correctly

1. Блокнот → «Сохранить как»
2. Тип файла: «Все файлы (*.*)»
3. Имя: `ORDER_SAFE_COPY_002.md`

## Forbidden

- .cdr, .art, CNC files
- Оригиналы заказов (не копии)
- Файлы из реальных рабочих папок
