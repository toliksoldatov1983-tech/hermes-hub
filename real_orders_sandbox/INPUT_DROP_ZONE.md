# Input Drop Zone

Date: 2026-06-17

## Правило

Пользователь **вручную** кладёт safe copy заказа в эту папку.

## Что можно положить

| Формат | Приоритет |
|--------|-----------|
| `.txt` | Высокий |
| `.md` | Высокий |
| `.json` | Средний |
| `.csv` | Средний |
| `.xlsx` | Средний |

## Что НЕЛЬЗЯ класть

- `.cdr` — Corel файлы
- `.art` — ArtCAM файлы
- CNC файлы
- Оригиналы заказов (ТОЛЬКО копии)

## Формат имени

```
safe_copy_YYYY-MM-DD_описание.txt
```
