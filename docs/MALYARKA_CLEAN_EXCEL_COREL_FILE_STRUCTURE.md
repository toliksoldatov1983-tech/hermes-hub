# Malyarka Clean: структура будущего Excel-файла для Corel

Дата: 2026-06-14

Серия:

```text
BATCH_SERIES_048_050_SAFE_EXCEL_COREL_EXPORT_PLANNING
```

## Назначение

Этот документ описывает структуру будущего Excel-файла для Corel.

Сейчас Excel-файл не создаётся. Это только план структуры.

## Формат файла

Будущий файл:

```text
.xlsx
```

Ориентировочное место будущего файла:

```text
E:\Hermes-Hub\outputs\COREL_EXPORT.xlsx
```

## Таблица

В будущей таблице должно быть 3 колонки:

```text
высота
ширина
количество
```

## Заголовки

Заголовков в Excel-файле быть не должно.

Первая строка должна быть пустая.

Данные начинаются после пустой первой строки.

## Какие строки включать

В Excel попадают только подтверждённые строки заказа.

Спорные строки не включаются.

Если заказ имеет `status = has_disputes` или `status = empty_or_invalid`, Excel-файл не готовится.

## Порядок строк

Порядок строк должен быть таким же, как в исходном заказе.

Export не должен сортировать строки и не должен менять порядок пользователя.

## Формат значений

В ячейках должны быть только числа:

- высота в миллиметрах;
- ширина в миллиметрах;
- количество.

Без лишнего текста, единиц измерения, комментариев и подсказок.

## Что не включать

В будущий Excel не включать:

- спорные строки;
- причины споров;
- площадь;
- стоимость;
- цены;
- ЛКМ;
- материалы;
- комментарии пользователя;
- Telegram/Vision/API-данные;
- данные старой Малярки.

## Что не входит сейчас

Сейчас не создаётся:

- Excel-файл;
- Corel-файл;
- export на диск;
- производственный файл.
