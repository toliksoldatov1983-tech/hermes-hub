## Рекомендация

Следующим крупным направлением рекомендуется **Excel/Corel export**.

Почему:

- локальная v0.1 уже умеет разобрать заказ;
- есть подтверждённые строки;
- есть площадь;
- есть внутренние Corel-строки;
- пользователю дальше нужен практический производственный слой.

## Важно

Сейчас это только рекомендация и план.

В этой серии:

- Excel/Corel export не реализуется;
- Excel-файлы не создаются;
- Corel-файлы не создаются;
- export на диск не делается.

## Что нужно перед реализацией

Перед Excel/Corel export нужен отдельный безопасный план:

1. Что именно должен получить пользователь.
2. Какой формат файла или структуры нужен.
3. Какие поля брать из текущего результата.
4. Как блокировать export при спорных строках.
5. Как проверять результат без риска для производства.
6. Какие файлы можно создавать.
7. Какие файлы нельзя трогать.

После плана нужно отдельное разрешение пользователя.
