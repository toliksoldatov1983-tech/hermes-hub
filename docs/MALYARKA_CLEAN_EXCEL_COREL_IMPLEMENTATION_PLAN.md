# Malyarka Clean: план будущей реализации Excel/Corel export

Дата: 2026-06-14

Серия:

```text
BATCH_SERIES_048_050_SAFE_EXCEL_COREL_EXPORT_PLANNING
```

## Назначение

Этот документ описывает будущую реализацию Excel/Corel export.

Сейчас код не пишется. Excel-файлы и Corel-файлы не создаются.

## Где создавать будущий код

Будущий код export можно будет создать внутри:

```text
E:\Hermes-Hub\projects\malyarka-clean
```

Предполагаемое место:

```text
E:\Hermes-Hub\projects\malyarka-clean\src\malyarka_clean_core\excel_corel_export.py
```

Точное имя файла нужно подтвердить в отдельном техническом пакете.

## Какие функции могут понадобиться

Будущие функции:

- подготовить данные для Excel из `corel_rows`;
- проверить, разрешён ли export;
- заблокировать export при спорном или пустом заказе;
- создать `.xlsx` только для clean-заказа;
- вернуть понятный результат: создан файл или export заблокирован.

## Какие сценарии проверить

Нужно проверить:

```text
clean -> Excel можно создать
has_disputes -> Excel заблокирован
empty_or_invalid -> Excel заблокирован
```

## Какие тесты нужны

Будущие focused tests:

- clean-заказ создаёт структуру Excel;
- Excel содержит только подтверждённые строки;
- первая строка пустая;
- заголовков нет;
- колонки идут в порядке: высота, ширина, количество;
- has_disputes блокирует export;
- empty_or_invalid блокирует export;
- спорные строки не попадают в Excel;
- парсер, площадь и dispute rules не меняются.

## Где может лежать будущий файл

Ориентировочный путь:

```text
E:\Hermes-Hub\outputs\COREL_EXPORT.xlsx
```

Файл можно будет перезаписывать, чтобы не плодить мусор.

## Как пользователь будет запускать будущий export

Возможный будущий порядок:

1. Пользователь открывает:

```text
notepad E:\Hermes-Hub\inputs\ORDER_INPUT.txt
```

2. Вставляет заказ и сохраняет.

3. Запускает будущую команду export.

4. Если заказ чистый, появляется:

```text
E:\Hermes-Hub\outputs\COREL_EXPORT.xlsx
```

5. Если заказ спорный или пустой, export не создаётся, а пользователь видит причину блокировки.

## Что нужно перед реализацией

Перед реализацией нужен отдельный технический пакет:

```text
Серия 051–053 — Реализация безопасного Excel/Corel export
BATCH_SERIES_051_053_SAFE_EXCEL_COREL_EXPORT_IMPLEMENTATION
```

И отдельное разрешение пользователя.

## Что не входит

В этот план не входит:

- написание кода;
- создание Excel-файла;
- создание Corel-файла;
- export на диск;
- Telegram;
- Vision;
- API;
- база данных;
- цены;
- стоимость;
- ЛКМ;
- материалы;
- производство;
- старая Малярка как активная система.
