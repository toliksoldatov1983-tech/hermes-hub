## Зачем нужен Telegram

Telegram нужен, чтобы пользователь мог отправить текст заказа привычным способом и быстро получить результат проверки.

Это может заменить ручной путь:

```text
ORDER_INPUT.txt -> ЗАПУСТИТЬ_ЗАКАЗ.cmd -> LAST_ORDER_RESULT.txt
```

Но первый Telegram-слой должен быть очень осторожным и минимальным.

## Минимальный безопасный Telegram-слой

На первом этапе бот должен делать только простые действия:

- принять текст заказа;
- передать текст в уже существующее ядро Malyarka Clean;
- показать результат проверки;
- показать статус заказа;
- показать площадь;
- показать спорные строки;
- если заказ `clean`, сообщить, что Excel можно создать;
- если заказ спорный или пустой, объяснить причину блокировки.

## Что бот должен делать на первом этапе

```text
Пользователь отправляет текст заказа
-> бот проверяет заказ через существующее ядро
-> бот возвращает понятный результат
```

Минимальный результат должен содержать:

- `status`;
- подтверждённые строки;
- спорные строки;
- `total_area_m2`;
- `export_blocked`;
- короткую подсказку, что делать дальше.

## Что бот НЕ должен делать на первом этапе

```text
Vision
database
external API
prices
cost
LKM
materials
production export
old bot.py as active source
```

Также бот не должен сам запускать рискованные действия без отдельного подтверждения.

## Excel на первом Telegram-этапе

На первом этапе достаточно, чтобы бот сказал:

```text
Заказ clean. Excel можно создать локальным запускателем.
```

Отправку Excel-файла через Telegram лучше вынести в отдельный будущий этап.

## Зоны риска

### Telegram token

Токен нельзя показывать в чате, документах или отчётах.

### Polling / live-запуск

Polling нельзя запускать без отдельного разрешения, потому что это реальное подключение к Telegram.

### .env

`.env` нельзя читать или менять без отдельного разрешения.

### Старый bot.py

Старый `bot.py` нельзя использовать как активную систему. Его можно рассматривать только после отдельного read-only аудита и только если пользователь отдельно разрешит.

### Telegram UX

Нужно заранее решить, какие команды и ответы будут у бота, чтобы он не путал пользователя.

### Отправка файлов

Отправка `COREL_EXPORT.xlsx` через Telegram — отдельный рискованный слой. Его нельзя включать в первый этап без отдельного плана.
