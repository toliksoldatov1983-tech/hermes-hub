## Что принято

Безопасный Excel/Corel export принят как рабочий локальный слой после v0.1.

Файл создаётся здесь:

```text
E:\Hermes-Hub\outputs\COREL_EXPORT.xlsx
```

## Условия создания Excel

Excel создаётся только если заказ чистый:

```text
status = clean
```

Дополнительные условия:

- спорных строк нет;
- `export_blocked = false`;
- есть готовые внутренние `corel_rows`.

## Когда export блокируется

Excel не создаётся, если:

- `status = has_disputes`;
- `status = empty_or_invalid`;
- есть спорные строки;
- заказ пустой или непонятный.

## Что попадает в Excel

В Excel попадают только подтверждённые строки.

Спорные строки не попадают в Excel.

Структура файла:

- `.xlsx`;
- первая строка пустая;
- заголовков нет;
- 3 колонки: `height`, `width`, `quantity`;
- числа без лишнего текста.

## Что не менялось

В серии реализации Excel/Corel export не менялись:

- парсер;
- расчёт площади;
- правила спорных строк.

## Чем этот Excel не является

Этот Excel — только технический файл для Corel.

Он не является:

- производственным расчётом стоимости;
- расчётом цены;
- расчётом ЛКМ;
- расчётом материалов;
- базой заказов;
- документом для клиента.
