# Malyarka Clean: локальная версия v0.1

Дата: 2026-06-13

Пакет:

```text
BATCH_039_LOCAL_V0_1_ACCEPTANCE
```

## Что считается локальной v0.1

Локальная версия v0.1 считается собранной как минимальная локальная проверка заказа без внешних сервисов.

В v0.1 есть:

- разбор текстового заказа;
- подтверждённые строки;
- спорные строки;
- расчёт площади по подтверждённым строкам;
- статусы `clean`, `has_disputes`, `empty_or_invalid`;
- ручной ввод заказа;
- ввод через `ORDER_INPUT.txt`;
- сохранение результата в `LAST_ORDER_RESULT.txt`;
- пользовательские `.cmd`-ярлыки;
- русская инструкция.

## Главный пользовательский сценарий

```text
notepad E:\Hermes-Hub\inputs\ORDER_INPUT.txt
E:\Hermes-Hub\ПРОВЕРИТЬ_ЗАКАЗ_ИЗ_ФАЙЛА.cmd
notepad E:\Hermes-Hub\outputs\LAST_ORDER_RESULT.txt
```

## Что v0.1 не делает

- не создаёт Excel-файл;
- не создаёт Corel-файл;
- не делает производственный экспорт;
- не запускает Telegram;
- не использует Vision;
- не подключает API;
- не использует базу данных;
- не считает цены, стоимость, ЛКМ и материалы;
- не запускает производство или CNC.

## Итог

v0.1 — это безопасная локальная основа: можно вручную дать текст заказа, получить понятный результат и сохранить его в `.txt`.
