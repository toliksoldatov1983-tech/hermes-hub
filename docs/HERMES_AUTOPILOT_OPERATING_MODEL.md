|------|-------|---------|

| 🔴 Красная | 0 | Hard stop |

## Счётчик задач

Зелёная задача = одно атомарное действие:
- Создание одного markdown-файла
- Обновление одного tracking-файла
- Запуск одного pytest-сьюта (только в разрешённой зоне)

Групповые действия (создание 2+ файлов за один вызов write_file/patch) считаются одной задачей.
