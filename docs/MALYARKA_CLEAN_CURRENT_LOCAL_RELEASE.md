# Malyarka Clean — текущая локальная версия

Дата: 2026-06-14

Серия:

```text
Серия 061–063 — Упрощение пользовательской инструкции и контрольная проверка локальной версии
BATCH_SERIES_061_063_USER_GUIDE_AND_LOCAL_RELEASE_CHECK
```

## Что собрано

Текущая локальная версия Malyarka Clean собрана как рабочая локальная точка с Excel/Corel export.

Это не Telegram, не Vision, не API, не база данных и не производственный export.

## Главный запускатель

Основной запуск:

```text
E:\Hermes-Hub\ЗАПУСТИТЬ_ЗАКАЗ.cmd
```

## Входной файл заказа

Пользователь пишет заказ сюда:

```text
E:\Hermes-Hub\inputs\ORDER_INPUT.txt
```

Быстро открыть файл заказа можно через:

```text
E:\Hermes-Hub\ОТКРЫТЬ_ФАЙЛ_ЗАКАЗА.cmd
```

## Текстовый результат

После запуска результат сохраняется сюда:

```text
E:\Hermes-Hub\outputs\LAST_ORDER_RESULT.txt
```

Быстро открыть результат можно через:

```text
E:\Hermes-Hub\ОТКРЫТЬ_ПОСЛЕДНИЙ_РЕЗУЛЬТАТ.cmd
```

## Excel для Corel

Если заказ чистый, Excel для Corel создаётся или обновляется здесь:

```text
E:\Hermes-Hub\outputs\COREL_EXPORT.xlsx
```

Excel создаётся только при статусе:

```text
clean
```

## Блокировка Excel

Excel не создаётся и не обновляется, если заказ получил статус:

```text
has_disputes
empty_or_invalid
```

В этих случаях нужно открыть `ORDER_INPUT.txt`, исправить заказ и снова запустить:

```text
E:\Hermes-Hub\ЗАПУСТИТЬ_ЗАКАЗ.cmd
```

## Что уже умеет локальная версия

- читать заказ из обычного `.txt`;
- разбирать простые размеры;
- отделять подтверждённые строки от спорных;
- считать площадь подтверждённых строк;
- сохранять читаемый текстовый результат;
- готовить Excel для Corel только для чистого заказа;
- блокировать Excel при спорных или пустых заказах.

## Что не входит в текущую локальную версию

```text

Vision
API
database
prices
cost
LKM
materials
production export
CNC
old Malyarka as active system
```
