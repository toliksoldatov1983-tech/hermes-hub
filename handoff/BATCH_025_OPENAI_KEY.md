## Проблема

## Что сделать

1. У тебя есть рабочий OpenAI ключ (ты через него работаешь)
2. Положи его в `/etc/malyarka-telegram-bot.env`:
   ```
   OPENAI_API_KEY=твой_ключ
   ```
3. Перезапусти бота
4. Проверь: отправь боту любое фото — должно распознать
