# BATCH_015 — Agent Implementation

Дата: 2026-06-24
Для: Codex

## Цель

Реализовать 4 агента Malyarka для сервера.

## Агенты

### 1. Corel Agent (`malyarka_core/exports/corel.py`)

Уже есть заготовка. Доделать:
- Генерация Corel Excel из OrderDraft
- Отдельные листы: подтверждённые, спорные
- Колонки: №, высота, ширина, количество, площадь м², примечание
- Итого: общая площадь, общее количество

### 2. Malyarka File Agent (`malyarka_core/exports/malyarka_file.py`)

Уже есть заготовка. Доделать:
- Генерация Файла Малярки (.txt) из OrderDraft
- Русские заголовки: «подтверждено», «итого подтверждено», «спорно»
- Формат: каждая строка — высота x ширина x количество

### 3. Memory Agent (`malyarka_hermes/memory.py`)

Уже есть заготовка. Доделать:
- Сохранение результатов заказа в лог
- Сохранение контекста диалога
- read-only, без БД

### 4. Diagnostics Agent (`malyarka_ai/diagnostics.py`)

Уже есть заготовка. Доделать:
- Проверка что все модули импортируются
- Проверка что парсер работает на тестовых данных
- Команда `/health` в боте

## Что сделать по каждому

1. Прочитать текущий файл на сервере
2. Дописать до рабочего состояния
3. py_compile проверка
4. Подключить в handlers.py (команды, кнопки)
