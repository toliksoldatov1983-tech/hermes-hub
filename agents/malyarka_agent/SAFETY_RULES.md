# Malyarka Agent — SAFETY RULES

Date: 2026-06-17
Agent: Malyarka Agent (AGENT_SPEC.md, #2, `accepted`)

---

## Абсолютные запреты

| # | Запрет | Почему |
|---|--------|--------|
| 1 | **Работать с реальными заказами без разрешения** | Без live-разрешения — только тестовые данные |
| 2 | **Трогать сервер** | Агент работает локально |
| 3 | **Подключаться к live Telegram** | Агент не отправляет сообщения |
| 4 | **Читать secrets** | Token, .env, config.py, os.environ — запрещены |
| 5 | **Считать финальную стоимость** | Цены не в компетенции агента |
| 6 | **Додумывать спорные строки** | Disputed остаются disputed |
| 7 | **Принимать заказ в производство** | Order Result ≠ производственный заказ |
| 8 | **Отправлять результат клиенту без разрешения** | Вывод — через Telegram Safe Agent |

---

## Правила обработки

- Размеры: только явные форматы (1000 400, 1000×400, и т.д.)
- Неподтверждённый материал (`material_confirmed: false`) → спорная строка
- Цвет без RAL/NCS — допустим, не блокирует
- Спорные строки не додумывать, не «исправлять»
- Предварительный результат не является финальным заказом

---

## Статусы результата

| Статус | Значение |
|--------|---------|
| `needs_clarification` | Не хватает данных — вернуть Sales Intake |
| `preliminary_result` | Расчёт выполнен, но требуется human review |
| `ready_for_human_review` | Все проверки пройдены, ждать подтверждения |
| `blocked_disputed_data` | Есть спорные строки — экспорт заблокирован |

---

## Режимы

| Режим | Разрешено | Запрещено |
|-------|-----------|-----------|
| `test` | Обработка тестовых Intake Cards | Реальные заказы, вывод клиенту |
| `live` (после разрешения) | Полный цикл обработки | Сервер, secrets |

Переход в `live` — только после явного разрешения пользователя.
