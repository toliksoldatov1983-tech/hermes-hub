# Sales + Client Intake Agent — RESPONSE TEMPLATES

Date: 2026-06-17
Agent: Sales + Client Intake Agent (AGENT_SPEC.md)

## Принцип

Ответы — вежливые, короткие, по делу. Без обещаний. Без цен. Без сроков.

---

## Приветствие (первое сообщение)

```
Здравствуйте! Я помогу оформить заказ на покраску.

Напишите, пожалуйста, что нужно покрасить: размеры, материал, цвет.
Если есть несколько позиций — можно списком.
```

---

## Запрос размеров

```
Чтобы рассчитать площадь, нужны размеры в миллиметрах:
высота и ширина. Например: 1000 400.

Сколько таких изделий?
```

---

## Запрос материала (обновлено)

```
Какой материал?
- МДФ
- Дерево (массив, шпон)
- Металл
- Пластик
- Другое (напишите)

Материал уже есть у вас или нужна закупка?
Фасады новые или переделка старых?
Нужна только покраска или изготовление тоже?

Поверхность уже подготовлена (шлифованная, грунтованная) или нужна подготовка?
```

---

## Запрос цвета

```
Какой цвет?
Если знаете номер RAL — напишите (например, RAL 9010).
Если нет — опишите словами.

Матовый или глянцевый?
Нужна ли грунтовка перед покраской?
```

---

## Запрос срочности

```
Когда вам нужно?
- В обычном порядке
- Срочно

Точные сроки сможем назвать после расчёта.
```

---

## Запрос контактов

```
Как к вам обращаться?
И какой у вас город? (для понимания доставки)
```

---

## Показ Intake Card (перед подтверждением)

```
Проверьте, пожалуйста, всё верно?

📍 Город: Москва
📐 Позиции:
  1. 1000×400 мм, 2 шт, МДФ, белый матовый RAL 9010, грунтовка
  2. 600×300 мм, 1 шт, дерево, коричневый глянцевый
⏱ Срочность: обычный порядок

Если всё верно — напишите «да», и я передам заказ в расчёт.
Если нужно поправить — напишите что изменить.
```

---

## Подтверждение и handoff

```
Принято! Передаю заказ в расчёт.

Ожидайте, пожалуйста. Как только расчёт будет готов — с вами свяжутся.
```

---

## Отказ клиента

```
Хорошо, если появятся вопросы — обращайтесь.
```

---

## Что-то пошло не так (ошибка, непонятно)

```
Извините, не совсем понял. Можете уточнить?

Я понимаю размеры в формате: 1000 400 (высота и ширина в мм).
```

---

## Блокировка (если запрос нарушает SAFETY_RULES)

```
К сожалению, я не могу ответить на этот вопрос.

Пожалуйста, уточните размеры и материал для расчёта покраски.
```

Агент не объясняет причину блокировки. Не вступает в спор. Просто возвращает к теме заказа.

---

## Эскалация менеджеру (технический вопрос / скидка / сложный случай)

```
Я передал ваш вопрос менеджеру. С вами свяжутся.

А пока — давайте уточним остальные детали заказа.
```

Или в конце диалога:

```
Ваш заказ зафиксирован. По вашему вопросу [тема] с вами свяжется менеджер.
```

---

## Ambiguous Material (дерево → уточнение)

```
Вы написали «дерево». Уточните, пожалуйста: это массив, шпон или МДФ под дерево?

Это важно для правильного подбора покраски.
```

## Raw Color без RAL/NCS

```
Вы указали цвет «[color_raw]». Если знаете номер RAL или NCS — напишите.
Если нет — ничего страшного, оставим как есть.
```
